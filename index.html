Yelp Admin Tool

yelptool.herokuapp.com

My Yelp Tool application was designed as an admin tool for the support team at Sprinklr to pull business information from Yelp's public API. The user uploads a plain text file containing comma separated Yelp Buisness IDs. Once these IDs are submitted, the user can watch the progress of the tool in the NodeJS console and a CSV is automatically downloaded with all results upon completion.

Built using NodeJS with Express and Bootstrap
