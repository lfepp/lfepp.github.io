Count to Pi

www.counttopi.com

My Count to Pi application tests a user's knowledge of pi to see how many digits they can recite. The user enters digits of pi into a form field until one is incorrect. A leaderboard of the top 20 all-time scores is then displayed.

Connecting Congress

www.connectingcongress.com

My Connecting Congress application allows a user to easily find the social media and email information for their local representatives in Congress. A user can enter their zip code to find all of their representatives in the House and Senate.

Yelp Admin Tool

www.github.com/lfepp/yelp

My Yelp Tool application was designed as an admin tool for the support team at Sprinklr to pull business information from Yelp's public API. The user uploads a plain text file containing comma separated Yelp Buisness IDs. Once these IDs are submitted, the user can watch the progress of the tool in the NodeJS console and a CSV is automatically downloaded with all results upon completion.
