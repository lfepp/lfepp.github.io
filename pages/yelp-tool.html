Yelp Admin Tool

My Yelp Admin Tool was originally built in a single day due to an urgent need for business information available via the Yelp API. The tool was designed to be used by a large team so ease-of-use was a high priority with very little time to design a user experience. The tool ingests a text file with any number of comma separated Yelp Buisness IDs and responds with a CSV file of the Yelp API responses.
